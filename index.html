Logic-Based Calculator

This calculator is designed as a brain-training tool rather than a traditional calculator. The design intentionally removes the 0 button and . to encourage logical thinking and problem-solving. Users must generate the value *0* by applying mathematical operations such as subtraction (for example, 1−1 or 2−2).

The goal of this calculator is to train the user’s "analytical mindset, mathematical reasoning, and creativity" when solving simple arithmetic problems. By limiting direct input and requiring users to construct certain numbers through operations, the calculator promotes deeper thinking instead of relying on straightforward typing.

The application is built using "HTML, CSS, and JavaScript. It features a clean user interface, a responsive grid layout for buttons, and interactive controls such as **ON, OFF, AC (clear all), C (delete last entry), and basic arithmetic operators*. The calculator evaluates expressions dynamically and displays results on a digital screen. Overall, this project demonstrates how simple programming concepts can be used to create a logic-training tool that strengthens problem-solving skills and computational thinking.
