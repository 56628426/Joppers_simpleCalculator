# Simple Web Calculator

A simple and responsive **web-based calculator** built using **HTML, CSS, and JavaScript**.
This project demonstrates basic front-end development concepts including DOM manipulation, event handling, and responsive layout using CSS Grid.

## Features

* Basic arithmetic operations: **Addition (+), Subtraction (-), Multiplication (×), Division (/)**
* **ON / OFF** functionality to simulate calculator power control
* **AC (All Clear)** button to reset the screen
* **C (Clear)** button to delete the last input
* **Responsive button layout** using CSS Grid
* Simple and colorful UI design
* Error handling for invalid calculations

## Technologies Used

* **HTML5** – Structure of the calculator
* **CSS3** – Styling and layout (Flexbox + Grid)
* **JavaScript (Vanilla JS)** – Logic and interactivity

## How It Works

The calculator listens for button clicks using JavaScript event listeners.
When a button is pressed, the corresponding value is displayed on the screen.
When the `=` button is pressed, the expression is evaluated using JavaScript and the result is displayed.

## Learning Purpose

This project was created to practice:

* DOM selection and manipulation
* Event listeners
* JavaScript logic for calculations
* CSS layout design
